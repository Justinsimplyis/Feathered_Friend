FLOW OF APP
----------------------
Navigation Flow
Splash Screen → Registration/Login Screen:
After the splash screen, navigate to either registration or login based on user choice.

Registration/Login Screen → Home Screen:
Successful registration or login takes the user to the home screen.

Home Screen → Map View:
Navigate to the map view to see hotspots and get directions.

Map View → Observations Screen:
Access the observations screen from the home screen or map view.

Observations Screen → Add Observation Screen:
Add new observations from the observations screen.

Add Observation Screen → Observations Screen:
Return to the observations screen after saving a new observation.

Home Screen → Settings Screen:
Access settings from the home screen.

Settings Screen → Home Screen:
Save settings and return to the home screen.

Observations Screen → Observation Details Screen (Optional):
View details of a specific observation from the list.
Profile Screen (Optional):

Accessible from the home screen or settings screen for managing profile details.
